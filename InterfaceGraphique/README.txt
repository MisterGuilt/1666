Pour compiler l'interface, il faut installer la SDL 1.2 version 64bits ainsi que l'extension sdl_ttf permettant d'afficher du texte.

Attention à un éventuel bug du compilateur MinGW sur to_string, qui nécessite le patch suivant: http://tehsausage.com/mingw-to-string

Cette version affiche le plateau de jeu avec les cartes des joueurs.

Il faut appuyer sur la touche échap pour faire avancer le programme.
